Hénon-Heiles System
  • X
    Value must be between 0.1 and 0.25
    Controls the X spatial coordinate of the system (0.1 - 0.25)
  • Y
    Value must be between 0.1 and 0.25
    Controls the Y spatial coordinate of the system (0.1 - 0.25)
  • PX
    Value must be between 0.1 and 0.25
    Controls the momentum of the X spatial coordinate of the system (0.1 - 0.25)
  • PY
    Value must be between 0.1 and 0.25
    Controls the momentum of the Y spatial coordinate of the system (0.1 - 0.25)
  • Time
    Value must be between 1000 and 100000
    The time span for which the integration of the Hénon-Heiles System will be performed (1000 - 100000)
