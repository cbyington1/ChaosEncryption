Lorenz System
  • Sigma
    Value must be between 5.0 and 10.0
    Controls the rate of convergence/divergence of the system's variables (5.0 - 10.0)
  • Rho
    Value must be between 20.0 and 40.0
    Controls the magintude of convergence/divergence of the system's variables (20.0 - 40.0)
  • Time
    Value must be between 20 and 100
    The time span for which the integration of the Lorenz System will be performed (20 - 100)
  • Iterations
    Value must be between 20 and 100
    The number of iterations or chaotic paths to generate in the Lorenz System (20 - 100)
  • X
    Value must be between 0.001 and 0.999
    Initial coordinate for the X spatial position (0.001 - 0.999)
  • Y
    Value must be between 0.001 and 0.999
    Initial coordinate for the Y spatial position (0.001 - 0.999)
  • Z
    Value must be between 0.001 and 0.999
    Initial coordinate for the Z spatial position (0.001 - 0.999)
