Rössler System
  • A
    Value must be between 0.1 and 0.3
    Controls how much the Y spatial variable influences the X spatial variable (0.1 - 0.3)
  • C
    Value must be between 5.0 and 6.0
    Controls how much the Z spatial variable influences the X spatial variable and Y spatial variable (5.0 - 6.0)
  • Time
    Value must be between 50 and 200
    The time span for which the integration of the Rössler System will be performed (50 - 200)
  • Iterations
    Value must be between 50 and 200
    The number of iterations or chaotic paths to generate in the Rössler System (50 - 200)
  • X
    Value must be between 0.001 and 0.999
    Initial coordinate for the X position (0.001 - 0.999)
  • Y
    Value must be between 0.001 and 0.999
    Initial coordinate for the Y position (0.001 - 0.999)
  • Z
    Value must be between 0.001 and 0.999
    Initial coordinate for the Z position (0.001 - 0.999)
